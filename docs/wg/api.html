createDataset

public Dataset createDataset(string name, string owner, string filename, boolean isFileName)

Creates a new dataset for a specific user.

Although the metadata of the dataset is returned, it is not yet imported. The status of the import can be checken using getImportStatus().

Files should be in valid CSV format.

Parameters:
name - The name of the dataset.
owner - The ID of the users that will own this dataset.
filename - The filename of the file to be uploaded, or the contents of the file.
isFileName - When true, the $file argument will be treated as a filename, otherwise it will be treated as the content of the CSV file itself.
Returns:
A newly created dataset object.
